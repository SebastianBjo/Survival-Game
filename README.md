# Forest Survival

## Description
Forest Survival is a 2D top-down survival game where the player must navigate and thrive in a realistic forest environment. The game features a day/night cycle, dynamic weather, temperature simulation, and a variety of resources to gather and crafting mechanics to keep the player alive.

## Installation
1. Clone the repository: `git clone https://github.com/your-username/forest-survival.git`
2. Navigate to the project directory: `cd forest-survival`
3. Install dependencies: `npm install`

## Running the Game
1. Start the development server: `npm start`
2. Open your web browser and visit `http://localhost:3000`

## Controls
- **Movement**: Use the arrow keys or WASD to move the player
- **Interact**: Press the space bar to interact with nearby resources or items
- **Inventory**: Press the 'I' key to open the inventory
- **Crafting**: Press the 'C' key to open the crafting menu

## Features
- Realistic 2D top-down forest environment
- Dynamic day/night cycle and weather system
- Temperature simulation that affects the player's health
- Hunger, thirst, and fatigue mechanics
- Illness and injury system
- Crafting system to create tools, shelter, and other items
- Harvestable resources such as food, water, and wood
- Wildlife that can be hunted for additional resources
- Minimalistic HUD displaying the player's vital signs
